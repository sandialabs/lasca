# file instructions

CEF_LaMnO: expressions of Gibbs energy and other quaantities, generated by "HydroGEN_CEF_general_current.py".

data: DFT data.

expt*: experimental data.

fit_CEF_lsm.py: script for fitting CEF.

HydroGEN_CEF_general_current.py: script for constructing CEF.

ho*: reduction enthalpy data from literature.

so*: reduction entropy data from literature.

O2_janaf: O2 free energy.

run_CEF.py: script for calling "HydroGEN_CEF_general_current.py".

results_wexpt: folder generated by "fit_CEF_lsm.py", storing the results about x(O)-p(O2).
